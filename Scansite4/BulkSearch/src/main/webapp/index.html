Scansite 4 Bulk Search New

Overview

Welcome to the Scansite 4 bulk search! This brand new feature gives you the opportunity to lift the use of Scansite to an entirely different level. The new feature becomes particularly handy, if you have multiple proteins, protein sequences or maybe even Motifs to analyze. Instead of waiting for each and every search after entering input data, it is now possible to start multiple searches at once, and leave or even turn off your computer. To run a bulk search, it is possible to enter data in multiple ways:


Direct Manual Input

The good old way of entering data manually is still possible. Now it is even extended. Within the bulk search you can enter multiple protein names or protein sequences once to analyze all of them.

File Input

Next to entering data manually, you might want to create a file in advance. This way you can prepare all the information at any place, even offline. To run the search you can simple upload a text file, which fulfills our input file criteria.
